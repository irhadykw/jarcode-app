Materi JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang didukung oleh semua browser web modern. Javascript merupakan salah satu teknologi inti dari pengembangan aplikasi berbasis web, bersama dengan HTML dan CSS yang mungkin telah Anda pelajari sebelumnya.

Javascript merupakan salah satu bahasa pemrograman web yang bisa membuat halaman website Anda lebih menarik serta lebih hidup. Bahasa pemrograman javascript itu dijalankan di perangkat pengunjung website atau situs, bukan di server, sebagaimana bahasa pemrograman PHP.

Javascript membuat website menjadi lebih responsif, jadi jika tidak menggunakan javascript tidak masalah, tapi akan sangat berpengaruh pada tampilan website. Jika mengibaratkan javascript seperti rumah, mungkin tidak ada masalah jika tidak menggunakan lampu atau tidak dipasang pintu, tetapi hal ini tentu akan menimbulkan kesan rumah akan gelap, suram, dan kurang menarik. Begitu pula dengan website, tidak adanya javascript membuat website kurang menarik.
