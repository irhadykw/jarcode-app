Materi HTML
(Hypertext Markup Language)

"HTML adalah bahasa markup standar untuk halaman Web. Dengan HTML Anda dapat membuat situs web Anda sendiri. HTML mudah dipelajari - Anda akan menikmatinya!"[w3schools.com]

"Demikianlah tiga kalimat pembuka yang ada pada website w3schools.com terkait HTML. Dari ketiga kalimat tersebut, dapat kita ketahui bahwa, HTML merupakan bahasa markup standar untuk membuat halaman web. Untuk mempelajari sangat mudah, dan pasti kita akan menikmati pelajaran HTML ini, apalagi bagi yang memiliki motivasi kuat untuk bisa membuat aplikasi berbasis web.

HTML5 adalah versi HTML terbaru telah disempurnakan. Secara teknis, HTML bukan bahasa pemrograman, melainkan bahasa markup.
