Materi CSS
(Cascading Style Sheet)

"CSS adalah bahasa yang menggambarkan style dari dokumen HTML. CSS menjelaskan bagaimana elemen HTML harus ditampilkan."[w3schools.com]

Dari kutipan tersebut di atas, dapat dipahami bahwa CSS, akronim dari Cascading Style Sheet, digunakan untuk mengontrol setiap elemen yang ada pada dokumen HTML dengan cara yang sederhana dan mudah, sehingga tampilan dokumen HTML lebih indah dan menarik.

Menggunakan CSS adalah keharusan dalam mengembangkan aplikasi berbasis web, atau dengan kata lain jika ingin menjadi desainer web - mendesain HTML dan CSS adalah keterampilan yang harus dimiliki.
